## Introduction

This project simulates a simple Memory Manager written in C#. It provides functionalities such as storing and retrieving data to/from RAM and ROM, performing basic arithmetic operations, and more.

## Features

- Store data in ROM
- Load data from ROM to RAM
- Perform arithmetic operations
- Execute custom commands
- And more...

## Prerequisites

- .NET 5 or above
- Visual Studio 2019 or a similar IDE

## Installation

1. Clone the repository:

   git clone https://github.com/yourusername/MemoryManager.git
2. Open the solution file `MemoryManager.sln` in Visual Studio.
3. Build the project to restore NuGet packages.
4. Run the application.

## Usage

Run the application and follow the on-screen instructions to interact with the Memory Manager. You can execute commands like `store`, `load`, `add`, etc., to manipulate data.
