Command Library

  • store <data>, <index> - Store data in ROM at the specified index.
  • load <index>, <accumulator> - Load data from ROM to the specified accumulator.
  • set <index>, <value> - Set the value of the specified accumulator and update ROM.
  • add <index1>, <index2>, <resultIndex> - Add values in two accumulators and store the result in the specified accumulator.
  • subtract <index1>, <index2>, <resultIndex> - Subtract the value of one accumulator from another and store the result in the specified accumulator.
  • multiply <index1>, <index2>, <resultIndex> - Multiply values in two accumulators and store the result in the specified accumulator.
  • divide <index1>, <index2>, <resultIndex> - Divide the value of one accumulator by another and store the result in the specified accumulator. (Division by zero is not allowed)
  • exchange <index1>, <index2> - Exchange the values between two accumulators.
  • jump <index> - Set the program counter to the specified index, effectively changing the execution order.
  • call <index> - Save the current program counter on the stack and set the program counter to the specified index, allowing for subroutine calls.
  • return - Pop the top value from the stack and set the program counter to that value, returning from a subroutine.
